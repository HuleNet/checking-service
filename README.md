# Checking Service 
Сервис для прогона кода по тестам, часть системы по проверке заданий по программированию для ВУЗа. Основан на чистой архитектуре.

## Технологии

- FastAPI
- PostgreSQL
- SQLAlchemy
- Pydantic
- Pytest
- MyPy
- Ruff
## Использование
Для установки и запуска проекта необходим Python v3.12+ и Poetry.

Установка зависимостей:
```sh
poetry install
```

Чтобы запустить проект, выполните команду:
```sh
poetry run app
```

Для запуска тестов выполните команду:
```sh
pytest
```
